== Description ==

This plugin will create an URL Record on your blog and will write all activity on your blog in main blog page.
The file report will found on <code>/wp-content/url-record/data/</code> you can analize it for prevent your blog from hacker or attacker.
This plugin is recomended for use if your hosting not use cpanel or etc.


== Installation ==

1. Upload url-record.php to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All record can be found on /wp-content/url-record/data/ directory
